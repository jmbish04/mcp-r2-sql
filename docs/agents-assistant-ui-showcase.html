Architectural Primitives

The core infrastructure elements powering the Agentic Cloud.

⚙

Dynamic Workers

The DynamicWorkerExecutor constructs purpose-built V8 sandboxes. Eliminates file system access and explicitly controls outbound fetches for zero-trust code execution.

🔒

Durable Object Facets

Allows Dynamic Workers to instantiate DOs with isolated SQLite databases. Enables platforms to run persistent, stateful code generated on-the-fly by AI.

📜

Code Mode MCP

Instead of thousands of discrete MCP tool calls, agents write a single TypeScript execution plan that orchestrates Cloudflare APIs, reducing token context by up to 80%.
